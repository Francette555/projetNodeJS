## ✨ Fonctionnalités

### Dashboard Principal
- Affichage des indicateurs clés :
  - Nombre total d'employés
  - Masse salariale totale
  - Salaire moyen
  - Salaires minimum et maximum

### Visualisation des données
- **Camembert** : Visualisation des statistiques globales
  - Total des salaires
  - Salaire maximum
  - Salaire minimum
- **Diagramme à barres** : Distribution individuelle des salaires par employé

### Gestion (à implémenter)
- Ajout / Modification / Suppression d'employés
- Recherche et filtrage
- Export des données

## 🛠 Technologies utilisées

### Frontend
- **React.js** 18.x - Bibliothèque UI
- **React Bootstrap** - Composants d'interface
- **Recharts** - Bibliothèque de graphiques
- **React Bootstrap Icons** - Icônes vectorielles
- **Axios** - Client HTTP

### Backend
- **Node.js** - Runtime JavaScript
- **Express.js** - Framework web
- **MySQL / MariaDB** - Base de données relationnelle

### Outils de développement
- **Create React App** - Outil de build
- **Nodemon** - Rechargement automatique

## 📦 Prérequis

Avant de commencer, assurez-vous d'avoir installé :

- [Node.js](https://nodejs.org/) (version 14.x ou supérieure)
- [npm](https://www.npmjs.com/) (version 6.x ou supérieure)
- [MySQL](https://www.mysql.com/) ou [MariaDB](https://mariadb.org/) (version 10.4+)
- [XAMPP](https://www.apachefriends.org/) (optionnel - pour environnement local)
